Weather App
This project features a weather app which gives the weather data of the user's current location. The project was built using vanilla JavaScript and the 'OpenweatherMap' api.

The app accesses the user's location using the navigator function of the client's browser. An error was also integrated into the application in the case where, the user denies the weather application access to the user's location.
